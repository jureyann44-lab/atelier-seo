## Pourquoi la vitesse est devenue cruciale pour le SEO

Depuis 2021, Google intègre les **Core Web Vitals** comme critère de classement officiel. Et les chiffres parlent d'eux-mêmes :

- Un site qui charge en 1 seconde convertit 3x plus qu'un site à 5 secondes
- 53% des visiteurs quittent un site mobile qui met plus de 3 secondes à charger
- Google privilégie les sites rapides dans ses résultats de recherche

Au Mans, la majorité des sites d'entreprises tournent encore sous WordPress avec des scores PageSpeed entre 30 et 60/100. C'est une opportunité concrète pour vous démarquer — sans budget publicitaire supplémentaire.

## Qu'est-ce qu'Astro et comment ça fonctionne

Astro est un framework web moderne qui bouleverse la manière de créer des sites performants.

### Le principe "Zero JavaScript par défaut"

WordPress classique charge 500 à 800 Ko de JavaScript, même si vous n'en avez pas besoin. Astro, lui, génère du HTML pur et ne charge du JavaScript que si c'est strictement nécessaire.

**Résultat concret :** un site Astro charge en moins d'1 seconde, là où un WordPress met 3 à 5 secondes.

### Pour qui c'est fait

Astro est parfait pour les sites vitrines d'entreprises, les portfolios, les blogs, les sites de documentation et les landing pages. En revanche, il est moins adapté aux applications web complexes ou aux interfaces nécessitant beaucoup d'interactivité en temps réel.

## Les Core Web Vitals expliqués simplement

Google mesure 3 indicateurs précis pour évaluer la qualité de votre site :

**LCP (Largest Contentful Paint)** — le temps de chargement du contenu principal visible. En dessous de 2,5 secondes c'est bien. Avec Astro, on est souvent sous la seconde.

**FID (First Input Delay)** — le temps avant que le site réponde à un clic ou un scroll. Avec Astro, c'est quasi-instantané car il n'y a pas de JavaScript bloquant.

**CLS (Cumulative Layout Shift)** — la stabilité visuelle pendant le chargement. Le HTML statique d'Astro donne une stabilité parfaite, zéro décalage.
