## Qu'est-ce qu'Astro et comment ça fonctionne

Astro est un framework web moderne qui bouleverse la manière de créer des sites performants.

### Le principe "Zero JavaScript par défaut"

**La différence avec WordPress ou React :**

WordPress classique : Charge 500-800 Ko de JavaScript (même si vous n'en avez pas besoin)

Astro : Génère du HTML pur, charge du JavaScript uniquement si nécessaire

**Résultat concret :** Un site Astro charge en < 1 seconde, là où un WordPress met 3-5 secondes.

## Les Core Web Vitals expliqués simplement

Google mesure 3 indicateurs précis :

### LCP (Largest Contentful Paint)

**Qu'est-ce que c'est :** Temps de chargement du contenu principal visible

**Mauvais :** > 4 secondes

**Bon :** < 2,5 secondes

**Astro :** Souvent < 1 seconde

### FID (First Input Delay)

**Qu'est-ce que c'est :** Temps avant que le site ne réponde à une interaction (clic, scroll)

**Mauvais :** > 300 ms

**Bon :** < 100 ms

**Astro :** Quasi-instantané (pas de JavaScript bloquant)

### CLS (Cumulative Layout Shift)

**Qu'est-ce que c'est :** Stabilité visuelle (les éléments ne bougent pas pendant le chargement)

**Mauvais :** > 0,25

**Bon :** < 0,1

**Astro :** HTML statique = stabilité parfaite
